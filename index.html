JavaScript Statements

In HTML, Javascript programs are executed by a web browser.

JavaScript statements are composed of: Values, Operators, Expressions, Keywords, and Comments. In the example below the statement tells the browser to write "Hello World" inside an HTML element with the id="hi"
document.getElementById("hi").innerHTML = "Hello World!";

Most JavaScript programs contain many JavaScript Statements.The Statements are executed, one by one, in teh same order as they are written.

JavaScript programs and statements are offten called the JavaScript code.

Semicolons seperate JavaScript Statements. Ensure to add a Semicolon a the end of each executable statement

b = 6; //Assigns the value of 6 to b
c = a + b; //Assigns the sum of a and b to c

When seperated by Semicolons, multiple statements on one line are allowed as show in this example:

In some examples on the web you might see examples without semicolons. However, ending statements with a semicolon is not required, but hightly recommended.

JavaScript Syntax

JavaScript Syntax is the set of rules, how JavaScriptprograms are constructed.

JavaScript Values

There are two types of JavaScript Syntax values: Fixed and Variable.

Fixed Values are called Literals.

Variable values are called Variables

JavaScript Literals

There are two important Syntax rules for fixed values:

  • Numbers are written with or without decimals: 10.20 or 1001
  • Strings are text written within double or single quotes: "John Doe" or "John Smith"

JavaScript Expressions

An expression is a combination of values, variable, and operators, which computes to a value.

The computation is called and evaluation. Example: 5 * 10 evaluates to 50

Expressions can alway contain a variable. Example: X * 10.

The values can be of various types, such as numbers and strings.Example: "John" + " " + "Doe" evalutes to "John Doe"

JavaScript Keywords

JavaScript keywords are used to identify actions to be performed.

The let keyword tells the browser to create variables.

Example:
let x, y;
x = 5 + 6;
y = x * 10;

The var keyword also tell the browser to create variables.

Example:
var x, y;
x - 5 + 6;
y = x * 10;

JavaScript Identifier/Name

These are names for JavaScript

Identifiers are used to name variables, keywords, and functions.

The rules for legal names are the same in most programming languages.

A name must begin with:

Numbers are not allowed as the first character in names, this way JavaScript can distinguish indentifiers from numbers
